# Doctor Booking App

A responsive doctor appointment booking web app built with React, Vite, Tailwind CSS, React Router, Axios, and React Toastify.

## Features
- Browse doctors
- View doctor details
- Book appointments
- Responsive UI
- Toast notifications
- Client-side routing

## Tech Stack
- React
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- React Toastify
