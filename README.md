# Doctor Booking

A doctor appointment booking web app built with React and Tailwind CSS.

## Tech Stack
- React 18, Vite, Tailwind CSS
- React Router DOM
- Axios
- React Toastify

## Getting Started

```bash
git clone https://github.com/Mino1826/doctor-booking.git
cd doctor-booking
npm install
npm run dev
Runs on http://localhost:5173
